TIENDA DE PLANTAS - DEMO BÁSICO (HTML + CSS + JS)

Estructura:
- index.html       -> Inicio con descripción simple
- productos.html   -> Catálogo de 6 productos + carrito (localStorage)
- login.html       -> Login y Registro (localStorage)
- styles.css       -> Estilos simples
- /js/app.js       -> Utilidades (moneda CLP, carrito, usuarios)
- /js/productos.js -> Render de productos y carrito
- /js/auth.js      -> Registro/Login en localStorage
- /assets/*.svg    -> Imágenes simples de plantas

Cómo usar:
1) Descomprime la carpeta.
2) Abre index.html en tu navegador.
3) Navega por el menú. El carrito y los usuarios quedan guardados en el navegador.

Notas:
- No hay backend; es solo para fines educativos.
- Puedes borrar datos desde las herramientas del navegador (localStorage).
